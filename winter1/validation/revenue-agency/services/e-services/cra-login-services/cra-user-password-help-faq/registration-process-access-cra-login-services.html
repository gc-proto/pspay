1. How do I register for a CRA user ID and password?

My Account for Individuals

You will have to complete two steps.

Step 1 – Provide personal information

  • Enter your social insurance number.
  • Enter your date of birth.
  • Enter your current postal code or ZIP code.
  • Enter an amount you entered on one of your income tax and benefit returns. Have a copy of your returns handy. (The line amount requested will vary. It could be from the current tax year or the previous one.) To register, a return for one of these two years must have been filed and assessed.
  • Create a CRA user ID and password.
  • Create your security questions and answers. You can also decide if you want a persistent cookie added to your computer, so you can access CRA Login Services using that same computer later without being asked for more identification.

After you complete step one of the registration process, you will have access to limited tax information on My Account. After you enter your CRA security code, you will have access to the full suite of services available in My Account, as well as the Auto-fill my return service.

Step 2 – Enter the CRA security code

To access your account, return to My Account for Individuals, select "CRA login," and enter your CRA user ID and password. When prompted, enter your CRA security code.

You can log into CRA Login Services with a Sign-in Partner. This option lets you log in with a user ID and password that you may already have, such as for online banking. For more information, see Sign-in Partners Help and FAQs.

  • Create a CRA user ID and password.

After you complete step one of the registration process, you will have access to some information in MyCRA. Once you have received your your CRA security code, we recommend entering it when you log in to MyCRA in order to access all of the services available.

Represent a Client

  • If you are a representative, you will be asked to: Enter the eight-character access code from your notice of assessment and your postal code or zip code.
  • Create a CRA user ID and password.
  • Create your security questions and answers. You can also decide if you want a persistent cookie added to your computer, so that you can access CRA Login Services using that same computer later without being asked for more identification.

To access the service anytime in the future, return to Represent a Client and log in using your CRA user ID and password.

You can also log into CRA Login Services with a Sign-in Partner. This option lets you log in with a user ID and password that you may already have, such as for online banking. For more information, see Sign-in Partners Help and FAQs.

Represent a Client—non-resident representative

If you are a non-resident representative living in the United States, you will be asked to:

  • Enter your non-resident representative number and your ZIP code.
  • Create a CRA user ID and password.
  • Create your security questions and answers. You can also decide if you want a persistent cookie added to your computer, so that you can access Represent a Client using that same computer later without being asked for more identification.

To access the service anytime, go to Represent a Client and log in using your CRA user ID and password.

Candidate profile

To register for the Candidate profile service, you will be asked to:

  • Complete the required fields for the candidate registration.
  • Create a CRA user ID and password.
  • Create your security questions and answers. You can also decide if you want a persistent cookie added to your computer, so you can access the Candidate profile service using that same computer later without being asked for more identification.

After you have completed the registration process, and your candidate profile, you will be able to apply to CRA jobs from anywhere with internet access.

You can also log into the Candidate profile service with a Sign-in Partner. This option lets you log in with a user ID and password that you may already have, such as for online banking. For more information, see Sign-in Partners Help and FAQs.

2. I have a Temporary Tax Number (TTN) or an Individual Tax Number (ITN). Can I still register?

Yes. You can register for our online services using a Temporary Tax Number (TTN) or Individual Tax Number (ITN). Enter your TTN or ITN when you are asked for a Social Insurance Number (SIN).

3. What do you mean by current postal code or ZIP code?

As part of your personal authentication information, you will have to enter the postal code or ZIP code of your current mailing address. If you live outside Canada and the United States, the postal code or ZIP code is not required. The postal code or ZIP code field is not case sensitive and may or may not include spaces or dashes.

4. How do I get a CRA security code?

If you have completed the registration process through My Account for Individuals, My Business Account, the MyCRA app, MyBenefits CRA app, or the CRA BizApp, you will be issued a CRA security code. The CRA security code has an indicated expiry date. Follow the provided instructions before the code expires, or you will have to contact the Individual tax enquiries to have a new CRA security code issued to you.

If you choose to have the code mailed, we will mail it to the address we have on file. Have you moved recently? If you are blind or partially sighted, you can ask to receive your personal correspondence from the CRA in an alternate format.

5. Where do I enter my CRA security code?

Once you receive your new CRA security code, return to the CRA login services page, and select the service you registered with. If you registered for a CRA user ID and password, select the CRA login option, or, if you registered using the Sign-in Partner service, select the Sign-in Partner option. Follow the online instructions, and enter your CRA security code when prompted.

6. How do I get a non-resident representative number (NRRN)?

A non-resident representative number (NRRN) is a number issued to non-resident individuals living in the United States who would like to use the Represent a Client service to access information and services on behalf of individuals and businesses. To apply for an NRRN, you must submit Form RC391, Application for a Canada Revenue Agency Non-Resident Representative Number (NRRN), and provide supporting documentation to prove your identity. We will mail you a letter with your NRRN and instructions on how to use your NRRN to register for Represent a Client.

7. Can I use the same user ID and password to access all the CRA login services?

Yes. After you register for a CRA login service, you can use the same CRA user ID and password to access all the CRA login services. However, for some services, more identity information may be needed.

8. I am visually impaired. Can I receive my CRA security code in an alternate format?

The CRA offers forms, publications and correspondence in multiple formats for persons with a visual impairment. These formats include Braille, e-text (CD), MP3, or large print. If you would prefer to receive the CRA security code letter and other personal correspondence from the CRA in an alternate format, call 1-800-959-8281 for more information.

9. How do I create my CRA user ID and password?

When you create your user ID and password, we recommend that you:

  • make them easy for you to remember and hard for others to guess;
  • avoid using personal information such as your name, social insurance number, mailing address, or email address; and
  • always keep this information secure and do not share it with anyone.

You will not be able to proceed with the creation of a user ID and password unless you follow these rules:

User ID

Your user ID must contain between 8 and 16 characters with no spaces, and can contain up to seven digits. The only special characters you can use are: dot (.), dash (-), underscore (_), and apostrophe (').

Note

Each user ID must be unique. You will receive a message if the user ID you create has already been used.

Password

Your password must contain between 8 and 16 characters, one upper-case letter, one lower-case letter, one digit, no space, and no accented characters. The only special characters you can use are: dot (.), dash (-), underscore (_), and apostrophe ('). You cannot use more than 4 consecutive, identical characters. The password and the confirm password must match.

10. I have forgotten or misplaced my CRA user ID and/or password. Is there any way that I can recover them?

If you have forgotten your CRA user ID for any of the following CRA login services, you can recover it.

  • My Account
  • My Business Account
  • Log in to MyCRA
  • Log in to MyBenefits CRA
  • Log in to CRA BizApp
  • Change my direct deposit
  • Change my address

To recover it, return to the service you want to use and select "CRA Login". On the CRA login page, select "Forgot your user ID?" and follow the instructions.

If you have forgotten your CRA user ID for any of the following CRA login services, you cannot recover it. You have to register again.

  • Represent a Client
  • Tax information web service (includes Auto-fill my return, T2 Auto-fill, and Express)

If you have forgotten your CRA user ID for the Candidate profile service, you have to complete the candidate profile recovery process to create a new CRA user ID. This will allow your new CRA user ID you register for during the recovery process, to be linked to your existing candidate profile.

Candidate profile recovery

If you are a new user, you will need to register before using the Candidate profile service.

CRA Password

If you have forgotten or misplaced your CRA password, you can create a new password, provided you respond correctly to the security questions you selected and answered when you registered.

To create a new password, return to the service you want to use and select "CRA Login". On the CRA login page, select "Forgot your password?" and follow the instructions.

11. How do I manage my CRA security options such as, revoke my CRA user ID, change my password, change my security questions and answers, etc.?

To manage any of the following CRA security options, you must first login to a service. Once you have logged in, you can access these options on the "CRA login and security options" page.

  • Change CRA user ID
  • Change CRA password
  • Change CRA security questions and answers
  • Update additional security feature preference
  • Revoke CRA user ID
  • View the Terms and conditions of use
  • View recent CRA login history

12. I want to change my password but I cannot find or see the "Last CRA login" page.

If you have ticked the box entitled "Do not show me this page again today" when you logged in today, the "Last CRA login" page will be hidden for the rest of the day. This means that the manage and view options, such as change your password, will only be available to you after midnight Eastern Time.

If you do not select "Do not show me this page again today", this page will be displayed to you each time you login.

13. Checklist

  • Have you moved recently? We need to know your current address before you can register for My Account for Individuals, My Business Account, MyCRA mobile app or MyBenefits CRA mobile app. If you have moved recently, call our Individual tax enquiries to change your address.
  • Get your personal information ready to register for My Account for Individuals, My Business Account, MyCRA mobile app or MyBenefits CRA mobile app.
  • Get your access code from your notice of assessment and your postal code or ZIP code ready to register for Represent a Client.
  • Apply for your non-resident representative number (NRRN) if you are a non-resident representative living in the United States in order to register for Represent a Client.
  • Make sure your operating system and browser configurations and settings are correct.
  • Confirm that the CRA login services are available by checking the hours of service.
